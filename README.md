# Offensive Security Labs & Write-ups

This repository serves as a centralized documentation hub for offensive security exercises, Capture The Flag (CTF) challenges, and penetration testing labs. It contains technical walkthroughs, command execution logs, and exploitation methodologies used to solve various security scenarios.

## Scope of Content

* **OverTheWire:** Linux system administration, bash scripting, and privilege escalation walkthroughs (e.g., Bandit).
* **TryHackMe:** Structured learning paths, network exploitation, and room write-ups.
* **Vulnerable Machines:** Boot-to-root challenges, vulnerability assessments, and exploit chains for independent boxes (e.g., HackTheBox, VulnHub).

The primary focus is on technical execution, detailing the specific tools, payloads, and command-line utilities required to compromise targets and retrieve flags.
